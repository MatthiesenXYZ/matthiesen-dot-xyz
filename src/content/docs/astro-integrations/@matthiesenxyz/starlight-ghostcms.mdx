Easily migrate your Ghost Site to Starlight, an Starlight Plugin that turns your Ghost server into a Headless CMS!

<Aside title="Fun Fact" type="tip">  This website is running on `starlight-ghostcms`! Just checkout the [`Blog`](/blog/)</Aside>

## Features:
- **Starlight**: Starlight is Everything you need to build a stellar documentation website. Fast, accessible, and easy-to-use.
- **Seamless Integration**: Starlight-GhostCMS is a seamless integration for your Ghost CMS & Starlight Docs Site
- **Fast**: Starlight-GhostCMS is fast, it turns your Ghost CMS into a headless server and uses Astro to Generate static websites that run on the Edge.
- **Easy to use**: Starlight-GhostCMS is easy to use, it uses the same Ghost API as your Ghost site, so you don't have to worry about setting up a new API.
- **Built with AIK**: [`astro-integration-kit`](https://github.com/florian-lefebvre/astro-integration-kit) is a set of tools and utilities for building integrations for Astro.
